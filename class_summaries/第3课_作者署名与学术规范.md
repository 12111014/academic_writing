# 第3课：作者署名与学术规范

## 主要内容

### 1. 作者署名问题
- **署名是研究生期间最实际的问题**，甚至有专门论文研究这个问题
- **署名规则**（不成文规定）：
  - 第一作者：承担60-70%的工作
  - 通讯作者：通常是导师/老板
  - 中间作者：做实验、收集数据等
  - 署名要提前说好，避免后续争议

### 2. 作者数量
- **传统**：1-4个作者
- **现代趋势**：大模型、交叉研究导致作者列表膨胀
  - 有些论文第一页写"XXX Team"，所有作者列在附录
  - 因为引用格式要求列出所有作者，太长不方便

### 3. 致谢（Acknowledgments）
- 贡献不足以成为作者，但值得感谢的人
- 可能包括：
  - 提供idea但未实现的人
  - 指出相关参考文献的人
  - 帮助改代码的人
  - 帮助画图的人
  - 提供设备、算力、数据的人
- **注意**：改代码很重要，但不一定是成为作者的条件

### 4. 基金号（Funding）
- 必须列出基金号（导师会提供）
- 博士生如果有自己的基金，也要列出
- 基金结题验收时会统计有基金号的论文数量

### 5. 知识产权
- **职务产出**：在学校工作期间的产出归学校
  - Paper相对自由（发表后是公共知识）
  - 专利、代码、数据都归学校
  - 如果要用于创业，需要向学校申请转让并交费

### 6. 学术不端行为

#### 抄袭（Plagiarism）
- **主动抄袭**：直接copy别人论文（严令禁止）
- **自抄袭**：抄自己之前发表的论文（也要避免）
  - 即使作者相同，每篇论文是独立个体
  - 要用引用，并修改文字表述

#### 避免抄袭的方法
- 必须加引用
- 修改文字表述（不能一字不差）
- 引用别人的图也要修改（改颜色、改结构等）

#### 案例：北京智源AI研究院
- 大模型综述论文被指抄袭
- 处理结果：相关章节作者和员工被开除
- 说明：一旦被抓，处理很严格

### 7. 引用的好处
- 不费时（LaTeX加引用很简单）
- 讨好小同行（特别是小领域期刊）
- 认识新朋友（学术圈是人情圈）
- 鼓励别人引用自己（互惠互利）

### 8. 实验室规则优先
- 所有规则最终要遵循实验室和导师的要求
- 学生不能单独投稿（需要导师同意）
- 投稿费用一般由导师出
- 导师可能不允许投某些期刊（如预警期刊）

## 关键要点
1. 署名问题要提前协商，避免后续争议
2. 改代码很重要，但不一定是成为作者的条件
3. 学校期间的产出归学校，特别是专利和代码
4. 自抄袭也要避免，要用引用并修改表述
5. 引用是学术圈的社交方式，要善用
6. 最终要遵循实验室和导师的规则
